## Project Description
This website was created for the CIS 195 Chapter 7 Lab.  
The site focuses on a calm, nature-inspired theme called **Quiet Trails**, designed to demonstrate semantic HTML, CSS Grid layout, and responsive design.

## Pages Included
- **Home (index.html):** Introduction to the Quiet Trails theme
- **Mindful Walking (page1.html):** Uses article and aside elements
- **Nature Views (page2.html):** Uses figure and figcaption elements

## Features
- Semantic HTML (header, nav, main, footer)
- CSS Grid layout
- External stylesheet
- CSS shadow and opacity
- Responsive design for mobile, tablet, desktop, and print
- GitHub Pages ready

## Instructions
1. Download or clone the repository  
2. Enable GitHub Pages  
3. Open `index.html` to view the homepage
